# SkillSync AI: An AI-Based Job Recommendation System 🎯

A University of Southern Mindanao project that uses Machine Learning to match users to careers based on the O*NET database.

## How to Run This on Your Laptop

**Step 1: Clone the repository**
Download this code to your computer and open the folder in VS Code (or your preferred editor).

**Step 2: Create a Virtual Environment**
Open your terminal and type:
`python -m venv .venv`

**Step 3: Activate the Virtual Environment**
* On Windows: `.venv\Scripts\activate`
* On Mac/Linux: `source .venv/bin/activate`

**Step 4: Install Dependencies**
Tell Python to install Streamlit, Pandas, Scikit-Learn, and Excel readers by running:
`pip install -r requirements.txt`

**Step 5: Run the App!**
`streamlit run app.py`
